About us

Welcome to Argupedia! We are an online debate forum that embraces open-mindedness.
We use Argumentation Frameworks in order to bring some structure to online discourse.

A user creates an argument e.g. "This website is the best".
If someone else disagrees then that user asks a "critical question" which attacks the original argument!
Now the first argument is "out" and the second argument / critical question is "in".
Someone else can defend the original argument by attacking the second argument / critical question which would change the labellings again.

A user can also submit an "alternate" argument.
For instance if a user submits the argument "We need world peace and argupedia is the best way to achieve it!"
Then another user could post an alternate argument "We need world peace and love is the best way to achieve it!"
These two arguments have the same goals but different methods to achieve them. Thus they are both undecided until users vote on which one should be "in" / winning.

There are also self-attacking arguments.
These are always considered "out". An example would be "Save money by spending it" or "If I know one thing, it's that I know anything".
